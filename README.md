# 💍 Indian Wedding Cost Analysis using Python

## 📌 Project Overview

This project analyzes an **Indian Wedding Dataset** using **Python, Pandas, NumPy, Matplotlib, and Seaborn** to understand wedding expenses, service categories, and cost distribution across different wedding types and locations. The analysis provides visual insights that help users compare wedding budgets and identify spending patterns for various wedding services.

## 🎯 Objectives

* Clean and preprocess the wedding dataset.
* Rename inconsistent column names for easier analysis.
* Perform Exploratory Data Analysis (EDA).
* Analyze wedding costs based on wedding type, place, and service categories.
* Visualize relationships between different wedding expenses.
* Build an interactive filtering system using user input.

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## 📊 Analysis Performed

The project includes:

* Data Cleaning and Column Renaming
* Missing Value Analysis
* Descriptive Statistics
* Correlation Matrix
* Pair Plot Analysis
* Pivot Table Analysis
* Interactive User-Based Filtering
* GroupBy Operations
* Mean and Sum Calculations

## 📈 Visualizations

Various visualization techniques were used to understand the data:

* Count Plot
* Bar Plot
* Line Plot
* Pie Chart
* Histogram
* Point Plot
* Violin Plot
* Strip Plot
* Scatter Plot
* Pair Plot
* Joint Plot
* KDE Plot

## 🔍 Key Insights

* Different wedding types have significant variations in overall wedding costs.
* Decoration, Photography, Entertainment, and Clothing contribute heavily to the total budget.
* Cost patterns vary depending on the wedding location.
* Entertainment preferences differ across decoration categories.
* Pair plots reveal relationships between major wedding expense categories.
* Distribution plots help identify spending trends and possible outliers.

## 💡 Conclusion

This project demonstrates how Python-based data analysis can transform raw wedding data into meaningful business insights. Using exploratory data analysis and multiple visualization techniques, the project enables better understanding of wedding budgeting, expense distribution, and customer preferences. The analysis can assist wedding planners, event organizers, and customers in making informed financial decisions while planning weddings.
